二、下载项目

1、pip方式

对于那些发布在PyPI（Python Package Index，Python包索引，包管理平台）上的项目。

PyPI是Python官方的第三方软件仓库，就像手机上的应用商店。

很多项目会把稳定版本发布到相应的包管理平台：

  • Python：PyPI（可以用pip安装）
  • JavaScript：npm（可以用npm install安装），例如Node.js项目
  • C/C++：可能发布到vcpkg、conan等
  • Windows软件：可能直接提供exe安装包

  2. 用git clone下载项目

​ 执行后，Git会创建一个 buttonpad 文件夹，并把所有代码下载到里面。

3、pip\git\ZIP的区别

pip install git clone + pip install -e
安装位置 site-packages（系统目录） 你指定的任何位置
能否看到源码 难找（藏在Python目录里） 就在你面前
能否修改代码 可以但不推荐 可以，且立即生效
适合场景 只想用这个工具 想学习、修改、贡献代码
磁盘空间 小（只有必要文件） 大（包含所有文件）
版本控制 不能 可以用git管理

  • 用 git clone 下载的是完整源码，包括：

    • 所有历史提交记录（可以用 git log 查看）
    • 测试代码（tests/ 文件夹）
    • 示例程序（比如ButtonPad里的几十个小游戏）
    • 文档源码

  • 而直接 pip install 只安装运行所需的最小文件。

ZIP下载和git clone的区别：

  • ZIP下载：只能拿到当前版本的代码
  • git clone：可以随时用 git pull 获取项目的最新更新

四、安装项目

  1. 安装项目

  • 方式A、开发模式安装（推荐）

    • -e 是 --editable 的缩写，意思是”可编辑模式”

    • 这样安装后，你修改源码会立即生效，不用重新安装

  • 方式B、普通安装

    • 会把包复制到 Python 的 site-packages 里。（site-packages是Python的”第三方仓库”——一个专门存放所有通过pip安装的包的文件夹。你可以把它理解为手机的”应用安装目录”。）

    • 查看site-packages路径命令：

      python -c "import site; print(site.getsitepackages())"

    • 修改源码后需要重新安装才能生效

  1. 确认安装成功

五、运行项目

1、python -m方式

  • 关于-m模块：

    • 不用关心文件在哪

      • 用-m时，Python会自动在所有安装路径里搜索

      • 就像你在电脑上按Win键搜索程序，不用管它安装在C盘还是D盘

    • 区分导入和运行

      • 有些Python包既可以作为库被别的程序导入使用

      • 也可以作为独立的程序直接运行

      • -m就是告诉Python：”我要以独立程序的方式运行这个包”

3.1运行整个项目

  • 方法A、PyCharm底部的Terminal（相当于cmd），执行命令python -m buttonpad

  • 方法B、绿色运行按钮旁，编辑配置：

    • 点击 + → 选择 Python

    • 配置如下：

      • Name: ButtonPad (随便起个名字)

      • Script path: 这里不选，改为选 Module name

      • Module name: 输入 buttonpad

      • Working directory: 确保是项目根目录 E:\Project\PycharmProjects\gitproj\buttonpad

    • 点击 OK 保存

    • 点击绿色的▶️（配置Name）运行按钮，就会看到启动器菜单！
